# MGA: Memory-driven GUI Agent for Observation-centric Interaction

## TL;DR（一句话总结）
> MGA：三组件记忆架构（空间线索、结构化记忆、当前观察），将当前决策与历史产物分离，减少长时程 GUI 任务中的轨迹坍缩。

## 主要贡献
- 空间线索：提取的 UI 布局信息用于当前屏幕理解
- 结构化记忆：过去交互和任务状态的动态摘要
- 当前观察：对当前屏幕状态的直接感知
- 分离减少长时程任务中历史产物对上下文的污染
