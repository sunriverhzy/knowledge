# ProRe: A Proactive Reward System for GUI Agents via Reasoner-Actor Collaboration

## TL;DR
> ProRe: proactive reward system with a Reasoner–Actor collaboration architecture; Reasoner proactively queries Actor for additional state evidence before issuing rewards, improving reward quality for complex GUI tasks.

## Key Contributions
- Reasoner-Actor collaboration: reward model actively queries agent for more information before scoring
- Proactive evidence gathering: issues targeted questions to resolve ambiguity in partial observations
- Improves reward quality on complex multi-step GUI tasks compared to passive LLM judges
