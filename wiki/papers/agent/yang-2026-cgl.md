# CGL: Advancing Continual GUI Learning via Reinforcement Fine-Tuning

## TL;DR
> CGL: continual GUI learning framework combining progressive curriculum stages with RFT; prevents catastrophic forgetting while adapting to new GUI environments via elastic weight consolidation and replay mechanisms.

## Key Contributions
- Progressive curriculum: staged difficulty increase across continual GUI environments
- Elastic weight consolidation + experience replay to prevent catastrophic forgetting
- RFT with verifiable GUI rewards enables sample-efficient continual adaptation
