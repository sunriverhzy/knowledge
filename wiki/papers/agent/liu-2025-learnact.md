# LearnAct: Few-Shot Mobile GUI Agent with a Unified Demonstration Benchmark

## TL;DR
> LearnAct + LearnGUI: few-shot demonstration-based learning for mobile GUI agents; first comprehensive dataset (2,252 offline + 101 online tasks) for studying in-context demonstration use; ACL 2026 Findings.

## Key Contributions
- LearnGUI: first dataset for demonstration-based learning in mobile GUI agents (2,252 offline, 101 online tasks)
- LearnAct: unified framework retrieving and using few-shot demonstrations at inference time
- Significantly improves accuracy on complex/rare scenarios; enables personalized mobile agents
- ACL 2026 Findings
