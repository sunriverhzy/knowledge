# Code2World: A GUI World Model via Renderable Code Generation

## TL;DR（一句话总结）
> Code2World：将 GUI 屏幕状态表示为可渲染代码（HTML/CSS/JS）而非像素阵列；大幅提升 world model 的可压缩性、可编辑性和 sim-to-real 保真度。

## 主要贡献
- 可渲染代码表示：GUI 状态表示为 HTML/CSS/JS 而非像素
- 大幅减小 world model 体积同时保留语义内容
- 代码状态可被编辑/修改以符号化模拟动作效果
- 对语义 UI 内容的 sim-to-real 保真度高于像素级预测
