# SmartSnap: Proactive Evidence Seeking for Self-verifying Agents

## TL;DR
> SmartSnap: self-verifying agent that proactively captures targeted screenshots (snaps) to gather evidence for action verification rather than relying on passive observation of the current screen.

## Key Contributions
- Proactive evidence seeking: agent decides when and where to take targeted screenshots for verification
- Self-verification: agent checks its own action outcomes using gathered evidence before proceeding
- Reduces error propagation in long-horizon GUI tasks through proactive state confirmation
