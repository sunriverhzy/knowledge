# Mobile-Agent-v3.5: Multi-platform Fundamental GUI Agents

## TL;DR（一句话总结）
> GUI-Owl-1.5：多平台 GUI agent（2B–235B），带 Hybrid Data Flywheel、MRPO（多平台 RL）、工具/MCP 使用和记忆；在 20+ benchmark 上 SOTA，包括 OSWorld 56.5%，AndroidWorld 71.6%，ScreenSpotPro 80.3%。

## 主要贡献
- GUI-Owl-1.5：从 2B 到 235B 的 instruct/thinking 模型家族；云端-边缘协作
- Hybrid Data Flywheel：模拟环境 + 云沙箱用于 UI 理解和轨迹生成
- MRPO：多平台 RL 算法，缓解平台冲突和长时程训练效率低
- OSWorld 56.5%，AndroidWorld 71.6%，WebArena 48.4%，ScreenSpotPro 80.3%，OSWorld-MCP 47.6%
