# UI-AGILE: Advancing GUI Agents with Effective Reinforcement Learning and Precise Inference-time Grounding

## TL;DR
> Dense distance-based grounding rewards during training + cropping-based grid resampling at inference; 23% improvement on ScreenSpot.

## Key Contributions
- Distance-based reward: r = max(0, 1 - dist/threshold); dense gradient
- Cropping-based resampling: zoom into predicted region for small element precision
- 23% accuracy improvement on ScreenSpot at publication
