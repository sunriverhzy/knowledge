# ProRe: A Proactive Reward System for GUI Agents via Reasoner-Actor Collaboration

## TL;DR（一句话总结）
> ProRe：Reasoner-Actor 协作架构的主动奖励系统；Reasoner 在给出奖励前主动向 Actor 查询额外状态证据，提升复杂 GUI 任务的奖励质量。

## 主要贡献
- Reasoner-Actor 协作：奖励模型在打分前主动向 agent 查询更多信息
- 主动证据收集：提出有针对性的问题以解决部分观察中的歧义
- 相比被动 LLM judge 提升复杂多步 GUI 任务的奖励质量
