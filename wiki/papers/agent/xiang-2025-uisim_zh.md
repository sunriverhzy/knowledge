# UISim: An Interactive Image-based UI Simulator for Dynamic Mobile Environments

## TL;DR（一句话总结）
> UISim：轻量级图像 mobile UI 模拟器，绕过昂贵的 Android 模拟器；以低资源成本支持大规模并行 mobile GUI RL 训练。

## 主要贡献
- 图像 simulator：无需完整 Android 模拟即可将 mobile UI 状态渲染为图像
- 低资源成本：每个环境实例比完整 Android 模拟器便宜 10-100 倍
- 支持 mobile GUI RL 训练的大规模并行
- 覆盖包括转换和动画在内的真实 mobile 应用动态
