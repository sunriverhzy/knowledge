# OS-HARM: A Benchmark for Measuring Safety of Computer Use Agents

## TL;DR
> OS-HARM: first comprehensive safety benchmark for computer-use agents; measures harm from accidental actions (data deletion, unintended purchases) and adversarial UI attacks on real OS tasks.

## Key Contributions
- First safety benchmark specifically for computer-use agents in real OS environments
- Measures accidental harm: data loss, financial actions, privacy violations from agent mistakes
- Adversarial harm: UI injection attacks, malicious popups exploiting agent vulnerabilities
- Provides taxonomy of CUA failure modes and safety evaluation methodology
