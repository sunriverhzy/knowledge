# PAL-UI: Planning with Active Look-back for Vision-Based GUI Agents

## TL;DR
> PAL-UI: GUI agents that actively retrieve past observations (snapshots) when current state requires historical context; dual-level summarization agent + dedicated retrieval tool; ICLR 2026.

## Key Contributions
- Active look-back: agent decides when it needs to retrieve past screenshots/observations
- Dual-level summarization: observation-level cues + action-level outcome summaries
- Dedicated retrieval tool: efficient search over past states by semantic similarity
- ICLR 2026 accepted; improves long-horizon GUI task performance
