# VSC-RL: Advancing Autonomous Vision-Language Agents with Variational Subgoal-Conditioned Reinforcement Learning

## TL;DR
> Variational subgoal-conditioned RL for GUI agents: learns subgoal representations with SGC-ELBO objective; mobile and web GUI tasks.

## Key Contributions
- SGC-ELBO: variational objective for learning subgoal representations
- Conditions policy on inferred latent subgoals for long-horizon tasks
- Subgoal representations improve credit assignment in sparse-reward GUI settings
