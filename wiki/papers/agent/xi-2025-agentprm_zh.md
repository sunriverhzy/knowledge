# AgentPRM: Process Reward Models for LLM Agents via Step-wise Promise and Progress

## TL;DR（一句话总结）
> AgentPRM：使用 Promise（成功潜力）和 Progress（状态改进）信号的 LLM agent 过程奖励模型；为多步 GUI agent 训练提供稠密 credit assignment。

## 主要贡献
- Promise 信号：从当前步骤估计未来任务成功概率
- Progress 信号：衡量相比上一步的任务状态改进
- 结合两者提供稠密 credit assignment，无需完整轨迹结果
