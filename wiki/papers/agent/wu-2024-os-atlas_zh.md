# OS-ATLAS: A Foundation Action Model for Generalist GUI Agents

## TL;DR（一句话总结）
> 拥有 1300 万训练样本的大规模 GUI action foundation model；跨平台 GUI agent 的强 grounding 和动作生成基线。

## 主要贡献
- 横跨 web、mobile、desktop 的 1300 万 GUI action 样本
- 在 grounding benchmark（ScreenSpot 等）上的强基线
- 下游 GUI RL fine-tuning 的基础
