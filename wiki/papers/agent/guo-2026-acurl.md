# Autonomous Continual Learning of Computer-Use Agents for Environment Adaptation

## TL;DR
> ACuRL: autonomous curriculum RL framework that continually adapts computer-use agents to new environments with zero human data; multi-stage pipeline: exploration, context review, capability evaluation, curriculum task generation, RL training.

## Key Contributions
- ACuRL: multi-stage autonomous curriculum RL for environment adaptation of CUAs
- Zero human data: fully autonomous curriculum generation through environment exploration and capability evaluation
- Handles distribution shifts as apps evolve, update, or when agents encounter new software
- Directly addresses real-world deployment challenge: agents must work in user-specific environments
