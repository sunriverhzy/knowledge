# M2-Miner: Multi-Agent Enhanced MCTS for Mobile GUI Agent Data Mining

## TL;DR（一句话总结）
> M2-Miner：第一个基于 MCTS 的 mobile GUI agent 自动数据挖掘框架；InferAgent + OrchestraAgent + JudgeAgent 协作挖掘高质量轨迹；效率提升 64 倍；ICLR 2026。

## 主要贡献
- MCTS 引导轨迹探索：对应用状态空间的系统性覆盖
- 三 agent 协作：InferAgent（推理）+ OrchestraAgent（协调）+ JudgeAgent（质量过滤）
- 意图复用：复用发现的意图以减少冗余探索
- 相比先前方法效率提升 64 倍；ICLR 2026
