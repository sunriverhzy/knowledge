# GUI-AIMA: Aligning Intrinsic Multimodal Attention with a Context Anchor for GUI Grounding

## TL;DR
> GUI-AIMA: coordinate-free grounding by aligning the model's intrinsic multimodal attention with a context anchor; extracts which image patch the model is attending to rather than predicting explicit coordinates.

## Key Contributions
- Attention-only grounding: no coordinate token prediction; uses intrinsic model attention
- Context anchor: reference element guides spatial attention calibration
- Coordinate-free: naturally handles variable-resolution and varying UI layouts
