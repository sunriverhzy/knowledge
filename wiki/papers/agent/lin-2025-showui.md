# ShowUI: One Vision-Language-Action Model for GUI Visual Agent

## TL;DR
> Unified VLA (Vision-Language-Action) model for GUI agents; interleaved UI-guided visual token selection reduces computation while preserving grounding accuracy.

## Key Contributions
- Single model for grounding, planning, and action generation
- UI-guided visual token selection: focus on relevant UI regions
- Strong performance on ScreenSpot and GUI task benchmarks
