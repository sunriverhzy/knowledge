# History-aware Reasoning for GUI Agents

## TL;DR（一句话总结）
> GUI agent 的历史感知推理框架：结构化历史编码，捕获长轨迹中的 UI 状态转换、动作结果和错误恢复。

## 主要贡献
- 结构化历史编码：将过去 UI 状态和动作编码为紧凑可检索格式
- 历史条件推理：agent 明确推理过去的失败和成功
- 改善长时程 GUI 任务的错误恢复和任务完成
