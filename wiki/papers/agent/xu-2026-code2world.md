# Code2World: A GUI World Model via Renderable Code Generation

## TL;DR
> Code2World: represents GUI screen states as renderable code (HTML/CSS/JS) rather than pixel arrays; dramatically improves world model compressibility, editability, and sim-to-real fidelity.

## Key Contributions
- Renderable code representation: GUI states as HTML/CSS/JS rather than pixels
- Dramatically reduces world model size while preserving semantic content
- Code states can be edited/modified to simulate action effects symbolically
- Higher sim-to-real fidelity than pixel-level prediction for semantic UI content
