# ViMo: A Generative Visual GUI World Model for App Agents

## TL;DR
> ViMo: first visual world model for GUI agents that generates future screen observations as images (not text); Symbolic Text Representation (STR) for text content in images; enables visual planning for mobile app agents.

## Key Contributions
- First visual GUI world model: predicts next screen as image, not text description
- STR: symbolic text representation overlays text content with placeholders while preserving graphics
- Enables visual planning: agent can simulate future GUI states before acting
- ICLR 2026 accepted
