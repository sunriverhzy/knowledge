# AgentProg: Empowering Long-horizon GUI Agents with Program-guided Context Management

## TL;DR
> AgentProg: program-guided context management for long-horizon GUI agents; uses executable programs to compress and organize multi-step history into structured memory that persists across interactions.

## Key Contributions
- Program-guided context: agent writes mini-programs to summarize and compress interaction history
- Executable memory: stored programs can be re-executed to restore task context
- Significantly reduces context window usage while preserving task-critical information
