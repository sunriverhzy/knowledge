# WebWorld: A Large-scale World Model for Web Agent Training

## TL;DR
> WebWorld: large-scale web world model trained on millions of real web interactions; enables model-based RL by dreaming web state transitions; significantly improves web agent training efficiency.

## Key Contributions
- Large-scale web interaction dataset for training web world models
- Enables dream-based rollout bypassing real browser I/O latency
- Model-based RL with periodic real-environment re-anchoring
