# GUI-R1: A Generalist R1-style Vision-Language Action Model for GUI Agents

## TL;DR
> R1-style RLVR+GRPO for GUI agents: unified action space + binary verifiable rewards; SOTA on 8 benchmarks with only 3K training samples; emergent chain-of-thought reasoning.

## Key Contributions
- Unified GUI action space: click, swipe, keyboard all in one representation
- Binary verifiable rewards: +1 if coordinate in BBox, 0 otherwise
- GUI-R1-3K: only 3K curated samples → SOTA on ScreenSpot-Pro + 7 benchmarks
- Emergent System-2 reasoning: spontaneous internal monologues without explicit supervision
