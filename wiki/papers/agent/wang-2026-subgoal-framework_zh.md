# A Subgoal-driven Framework for Improving Long-horizon LLM Agents

## TL;DR（一句话总结）
> 长时程 LLM/GUI agent 的子目标驱动框架（MiRA）：通过里程碑规划和势能塑造奖励将抽象任务转化为具体子目标；直接适用于 web 导航。

## 主要贡献
- MiRA：长时程 agent 训练的里程碑规划 + 势能塑造
- 将任务分解为可验证子目标提供中间奖励信号
- 势能塑造确保奖励与原始任务目标的一致性
- 在需要 20+ 步规划的 web 导航任务上演示
