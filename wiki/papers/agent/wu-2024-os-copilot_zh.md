# OS-Copilot: Towards Generalist Computer Agents with Self-improvement

## TL;DR（一句话总结）
> 带自改进的通用 computer agent：FRIDAY 系统从过去失败和网页教程中学习，迭代改进 OS 任务性能。

## 主要贡献
- FRIDAY agent：用声明性 + 程序性记忆处理 OS 任务
- 从失败日志和在线文档中自改进
- 覆盖文件、终端、应用、web 任务的通用设计
