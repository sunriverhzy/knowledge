# Agentic Reward Modeling: Verifying GUI Agent via Online Proactive Interaction

## TL;DR（一句话总结）
> VAGEN：用主动 verifier agent（执行命令、检查状态）替换被动 LLM judge，为 GUI agent 训练产生无噪声 RLVR 奖励。

## 主要贡献
- VAGEN：配备交互工具的 verifier agent，自主规划验证策略
- 主动环境探测：执行系统命令验证副作用（不只是观察截图）
- 为没有简单程序化 verifier 的任务提供无噪声 RLVR
- 解决被动 LLM-as-judge 在复杂多步 GUI 验证中的局限
