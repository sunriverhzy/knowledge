# UI-R1: Enhancing Efficient Action Prediction of GUI Agents by Reinforcement Learning

## TL;DR（一句话总结）
> 基于规则的 RLVR + DAST（Difficulty-Adaptive Slow-Thinking）：根据任务难度自适应推理深度；跨平台 GUI grounding 与动作。

## 主要贡献
- DAST：难度自适应思考——复杂任务获得更多 CoT token
- 基于规则的 RL 奖励：动作类型验证 + 格式合规性
- 跨平台（D/M/W）GUI grounding 与多步任务执行
- 3B 模型规模；强数据效率
