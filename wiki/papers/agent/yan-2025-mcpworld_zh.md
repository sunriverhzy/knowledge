# MCPWorld: A Unified Benchmarking Testbed for API, GUI, and Hybrid Computer Use Agents

## TL;DR（一句话总结）
> MCPWorld：API-only、GUI-only 和 hybrid computer-use agent 的统一 benchmark；评测 agent 在 API 工具和 GUI 动作之间正确路由的能力；标准化跨范式 computer use 评测。

## 主要贡献
- 统一测试台：同一任务在 API、GUI 和 hybrid 条件下测试
- 评测路由智能：agent 何时该用 API vs GUI？
- 标准化跨不同 computer-use 范式的比较
