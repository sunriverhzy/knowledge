# AndroidWorld: A Dynamic Benchmarking Environment for Autonomous Agents

## TL;DR
> Dynamic Android RL benchmark: 116 parameterized task templates generating millions of unique tasks; ground-truth rewards via ADB/SQLite state inspection; prevents rote memorization.

## Key Contributions
- 116 parameterized task templates → millions of task variants
- Ground-truth rewards from direct ADB database queries (zero noise)
- Prevents overfitting via task randomization
- Definitive 'Gymnasium' for mobile RL training
