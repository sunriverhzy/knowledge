# ScreenSpot-Pro: GUI Grounding for Professional High-resolution Computer Use

## TL;DR
> GUI grounding benchmark for professional 4K/Retina screens; reveals that current VLMs struggle with small elements on high-resolution displays.

## Key Contributions
- Professional-grade screen grounding tasks (4K, UI-dense layouts)
- Reveals hallucination and precision failures of current VLMs
- Used by GUI-R1 and InfiGUI-G1 as evaluation target
