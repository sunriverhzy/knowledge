# BacktrackAgent: Enhancing GUI Agent with Error Detection and Backtracking Mechanism

## TL;DR（一句话总结）
> Generator-Verifier-Judger-Reflector 架构：检测错误、回退到安全状态、改进策略后重试；EMNLP 2025。

## 主要贡献
- 错误检测：Verifier 检查当前状态是否符合预期
- 回退：撤销最近动作回到最后已知良好状态
- Reflector：基于失败尝试信息更新策略
- EMNLP 2025；跨平台 GUI 任务
