# MobileGUI-RL: Advancing Mobile GUI Agent Through Reinforcement Learning in Online Environment

## TL;DR（一句话总结）
> Mobile GUI agent 的可扩展 online RL 框架：带效率奖励的 GRPO 训练 agent 发现简洁动作路径；解耦 CPU 模拟器与 GPU 推理实现并行训练；ICLR 2026。

## 主要贡献
- Mobile GUI 的可扩展 online RL：CPU 模拟器与 GPU 训练解耦实现大规模并行
- 带效率奖励的 GRPO：惩罚冗余动作，奖励更短的成功路径
- 证明 online RL 可以从零开始训练生产质量的 mobile GUI agent
- ICLR 2026 接收；在 AndroidWorld 上强性能
