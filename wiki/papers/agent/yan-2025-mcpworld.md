# MCPWorld: A Unified Benchmarking Testbed for API, GUI, and Hybrid Computer Use Agents

## TL;DR
> MCPWorld: unified benchmark for API-only, GUI-only, and hybrid computer-use agents; evaluates when agents correctly route between API tools and GUI actions; standardizes cross-paradigm computer use evaluation.

## Key Contributions
- Unified testbed: same tasks tested under API, GUI, and hybrid conditions
- Evaluates routing intelligence: when should agent use API vs GUI?
- Standardizes comparison across different computer-use paradigms
