# UISim: An Interactive Image-based UI Simulator for Dynamic Mobile Environments

## TL;DR
> UISim: lightweight image-based mobile UI simulator that bypasses expensive Android emulators; enables massively parallel mobile GUI RL training at low resource cost.

## Key Contributions
- Image-based simulator: renders mobile UI states as images without full Android emulation
- Low resource cost: 10-100x cheaper than full Android emulator per environment instance
- Enables massive parallelism for mobile GUI RL training
- Covers realistic mobile app dynamics including transitions and animations
