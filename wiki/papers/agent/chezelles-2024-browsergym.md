# The BrowserGym Ecosystem for Web Agent Research

## TL;DR
> Unified Gymnasium-style API aggregating WebArena, MiniWoB++, VisualWebArena, WebChoreArena; standardizes web-agent benchmark infrastructure.

## Key Contributions
- Single API for 4+ web benchmarks (WebArena, MiniWoB++, VWA, WebChoreArena)
- Handles Docker, browser state, DOM extraction behind a standard interface
- Enables cross-benchmark agent training and evaluation
