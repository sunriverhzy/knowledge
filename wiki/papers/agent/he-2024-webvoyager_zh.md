# WebVoyager: Building an End-to-end Web Agent with Large Multimodal Models

## TL;DR（一句话总结）
> 第一个使用 GPT-4V 的端到端多模态 web agent：将截图与文本交错；真实 web 任务上 55.7%。

## 主要贡献
- 第一个使用多模态（截图 + 文本）输入的端到端 web agent
- 使用 GPT-4V 进行视觉 web 理解和动作规划
- 建立了基于 LMM 的端到端 web agent 基线
