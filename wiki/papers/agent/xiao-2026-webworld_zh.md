# WebWorld: A Large-scale World Model for Web Agent Training

## TL;DR（一句话总结）
> WebWorld：在数百万真实 web 交互上训练的大规模 web world model；通过 dreaming web 状态转换实现 model-based RL；显著提升 web agent 训练效率。

## 主要贡献
- 用于训练 web world model 的大规模 web 交互数据集
- 支持绕过真实浏览器 I/O 延迟的 dream-based rollout
- 带定期真实环境重新锚定的 model-based RL
