# PAL-UI: Planning with Active Look-back for Vision-Based GUI Agents

## TL;DR（一句话总结）
> PAL-UI：当前状态需要历史上下文时主动检索过去观察（快照）的 GUI agent；双层摘要 agent + 专用检索工具；ICLR 2026。

## 主要贡献
- 主动回看：agent 决定何时需要检索过去的截图/观察
- 双层摘要：观察级线索 + 动作级结果摘要
- 专用检索工具：按语义相似性高效搜索过去状态
- ICLR 2026 接收；改善长时程 GUI 任务性能
