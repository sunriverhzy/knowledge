# ProgRM: Build Better GUI Agents with Progress Rewards

## TL;DR（一句话总结）
> ProgRM：GUI agent 的过程奖励模型，自动从 demonstration 中提取中间里程碑并提供稠密进度估计，克服长时程奖励稀疏问题。

## 主要贡献
- 从成功 demonstration 自动提取进度里程碑，无需手工标注
- 与任务进度成比例的稠密中间奖励；克服仅终止奖励的稀疏性
- 与 GRPO 和 PPO 兼容；显著改善长时程 GUI 任务训练
