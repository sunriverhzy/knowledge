# ScreenAgent: A Vision Language Model-driven Computer Control Agent

## TL;DR
> VLM-driven desktop computer control via VNC pixel-stream; Plan-Act-Reflect self-correcting loop; accessibility-API-free approach for cross-platform robustness.

## Key Contributions
- Pure pixel-stream via VNC: no accessibility API dependency
- Plan-Act-Reflect loop: reflect on action outcomes to self-correct
- Cross-platform: works on any OS exposing VNC interface
