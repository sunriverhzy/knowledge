# MGA: Memory-driven GUI Agent for Observation-centric Interaction

## TL;DR
> MGA: three-component memory architecture (Spatial Cues, Structured Memory, Current Observations) that separates current decisions from historical artifacts to reduce trajectory collapse in long-horizon GUI tasks.

## Key Contributions
- Spatial Cues: extracted UI layout information for current screen understanding
- Structured Memory: dynamic summaries of past interactions and task state
- Current Observations: direct perception of present screen state
- Separation reduces context pollution from historical artifacts in long-horizon tasks
