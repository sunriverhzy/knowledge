# ProgRM: Build Better GUI Agents with Progress Rewards

## TL;DR
> ProgRM: process reward model for GUI agents that automatically extracts intermediate milestones from demonstrations and provides dense progress estimations, overcoming long-horizon reward sparsity.

## Key Contributions
- Automatically extracts progress milestones from successful demonstrations without manual labeling
- Dense intermediate rewards proportional to task progress; overcomes terminal-only reward sparsity
- Compatible with GRPO and PPO; improves long-horizon GUI task training significantly
