# History-aware Reasoning for GUI Agents

## TL;DR
> History-aware reasoning framework for GUI agents: structured history encoding that captures UI state transitions, action outcomes, and error recovery across long trajectories.

## Key Contributions
- Structured history encoding: encode past UI states and actions into a compact, retrievable format
- History-conditioned reasoning: agent explicitly reasons about past failures and successes
- Improves error recovery and task completion in long-horizon GUI tasks
