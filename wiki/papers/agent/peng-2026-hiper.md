# HIPER: Hierarchical Reinforcement Learning with Explicit Credit Assignment for LLM Agents

## TL;DR
> Hierarchical RL with explicit credit assignment: decomposes long-horizon task credit into structured subgoal rewards to reduce assignment ambiguity.

## Key Contributions
- Structured hierarchical reward decomposition for long-horizon GUI tasks
- Explicit credit assignment at subgoal level — not just terminal reward
- Reduces credit-assignment ambiguity for 20–200 step trajectories
