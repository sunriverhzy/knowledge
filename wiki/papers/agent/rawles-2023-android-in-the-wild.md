# Android in the Wild: A Large-scale Dataset for Android Device Control

## TL;DR
> 715K+ Android device interaction trajectories across diverse apps and tasks; used for offline GUI agent pretraining and offline RL initialization (DigiRL, Digi-Q).

## Key Contributions
- 715K+ interaction trajectories from real Android device usage
- Diverse apps, tasks, and UI layouts across versions
- Standard pretraining data for mobile GUI agents (DigiRL, Digi-Q)
