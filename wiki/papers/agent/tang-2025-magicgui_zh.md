# MagicGUI: A Foundational Mobile GUI Agent with Scalable Data Pipeline and Reinforcement Fine-tuning

## TL;DR（一句话总结）
> 带可扩展数据流水线（轨迹合成）+ RL fine-tuning 的 foundational mobile GUI agent；广泛覆盖 Android 应用和任务。

- 广泛覆盖：多样 Android 应用类别和任务类型
