# LearnAct: Few-Shot Mobile GUI Agent with a Unified Demonstration Benchmark

## TL;DR（一句话总结）
> LearnAct + LearnGUI：mobile GUI agent 的少样本 demonstration 学习；第一个研究上下文 demonstration 使用的综合数据集（2252 个离线 + 101 个在线任务）；ACL 2026 Findings。

## 主要贡献
- LearnGUI：第一个面向 mobile GUI agent demonstration 学习的数据集（2252 个离线，101 个在线任务）
- LearnAct：统一框架，推理时检索并使用少样本 demonstration
- 在复杂/罕见场景上显著提升准确率；支持个性化 mobile agent
- ACL 2026 Findings
