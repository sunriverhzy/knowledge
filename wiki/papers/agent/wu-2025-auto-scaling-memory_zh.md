# Auto-scaling Continuous Memory for GUI Agent

## TL;DR（一句话总结）
> GUI agent 的自动缩放连续记忆：根据任务复杂度动态调整记忆大小；通过轨迹 embedding 和检索构建低成本数据飞轮；支持可靠的长时程 GUI 自动化。

## 主要贡献
- 自动缩放记忆：根据任务复杂度信号动态调整记忆预算
- 连续轨迹 embedding：将 GUI 状态和动作编码为可搜索记忆
- 低成本数据飞轮：过去的成功轨迹提升未来任务性能
