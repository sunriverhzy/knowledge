# ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents

## TL;DR
> ClawGUI: end-to-end unified framework for GUI agent lifecycle covering training pipelines, evaluation harnesses, and deployment infrastructure across diverse platforms and applications.

## Key Contributions
- Unified framework: single codebase for training, evaluating, and deploying GUI agents
- Multi-platform support: web, desktop, mobile in one training/eval pipeline
- Standardizes GUI agent development workflow from data to deployment
