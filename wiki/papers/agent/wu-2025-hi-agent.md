# Hi-Agent: Hierarchical Vision-Language Agents for Mobile Device Control

## TL;DR
> Hierarchical Planner+Executor architecture: Planner generates semantic subgoals, Executor performs UI actions; GRPO with Foresight Advantage Function; 87.9% on AitW.

## Key Contributions
- Planner: generates semantic subgoals from high-level task description
- Executor: performs atomic UI actions guided by current subgoal
- Foresight Advantage Function: considers future task progress in advantage estimation
- 87.9% on AndroidInTheWild (SOTA at publication)
