# UltraCUA: A Foundation Model for Computer Use Agents with Hybrid Action

## TL;DR（一句话总结）
> 统一视觉 GUI primitive 与 API/tool action 的 Foundation CUA；训练在通用视觉 fallback 与快速确定性 API 执行间路由。

## 主要贡献
- 统一混合动作空间：低层 GUI 点击 + 高层 API 调用
- 学得的 GUI/API 模态路由
- 在大规模跨平台 computer-use 数据上训练
