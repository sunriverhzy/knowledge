# Agent Q: Advanced Reasoning and Learning for Autonomous AI Agents

## TL;DR
> MCTS-guided trajectory search for high-quality DPO training data: 'search is data'; improved Llama-3-70B web booking from 18.6% to 81.7%.

## Key Contributions
- Guided MCTS with value-model pruning for high-quality web trajectory search
- Self-critique mechanism: AI-driven state evaluation during tree search
- MCTS paths → DPO preference pairs for offline policy training
- 340% relative improvement on real web booking (OpenTable)
