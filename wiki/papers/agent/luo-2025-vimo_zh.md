# ViMo: A Generative Visual GUI World Model for App Agents

## TL;DR（一句话总结）
> ViMo：第一个以图像形式（而非文本）生成未来屏幕观察的 GUI agent 视觉 world model；引入 Symbolic Text Representation（STR）处理图像中的文本内容；支持 mobile app agent 的视觉规划；ICLR 2026。

## 主要贡献
- 第一个视觉 GUI world model：以图像而非文本描述预测下一屏幕
- STR：用占位符覆盖文本内容同时保留图形的符号文本表示
- 支持视觉规划：agent 在行动前可以模拟未来 GUI 状态
- ICLR 2026 接收
