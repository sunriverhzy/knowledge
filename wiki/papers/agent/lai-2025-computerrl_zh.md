# ComputerRL: Scaling End-to-end Online Reinforcement Learning for Computer Use Agents

## TL;DR（一句话总结）
> 带 API-GUI hybrid action space 的 computer-use agent 端到端 online RL；引入 Entropulse 对抗长时程任务中的熵坍缩。

## 主要贡献
- 正式的 API-GUI hybrid action paradigm：agent 在 GUI 坐标和 API 调用间选择
- Entropulse：RL+SFT 交替策略，维持策略熵
- 在真实 computer 环境上的端到端 online RL 训练
- 在 OSWorld 上相比先前方法有显著性能提升
