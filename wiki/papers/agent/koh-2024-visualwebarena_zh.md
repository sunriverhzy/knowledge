# VisualWebArena: Evaluating Multimodal Agents on Realistic Visual Web Tasks

## TL;DR（一句话总结）
> WebArena 的视觉扩展：910 个视觉 web 任务，正确动作依赖视觉理解（图片内容、视觉布局）而非仅文本；测试 web agent 中的视觉 grounding。

## 主要贡献
- 910 个需要对页面内容进行视觉理解的视觉 web 任务
- 用图像相关任务规格扩展 WebArena
- 多模态 web agent 评测的关键 benchmark
