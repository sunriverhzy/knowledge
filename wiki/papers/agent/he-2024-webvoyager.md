# WebVoyager: Building an End-to-end Web Agent with Large Multimodal Models

## TL;DR
> First end-to-end multimodal web agent using GPT-4V: interleaves screenshots with text; 55.7% on real-world web tasks.

## Key Contributions
- First end-to-end web agent using multimodal (screenshot + text) input
- Uses GPT-4V for visual web understanding and action planning
- Establishes end-to-end web agent baseline with LMMs
