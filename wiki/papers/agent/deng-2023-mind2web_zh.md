# Mind2Web: Towards a Generalist Agent for the Web

## TL;DR（一句话总结）
> Web agent 数据集 + 模型：2K+ 多样 web 任务及动作标注；语义 DOM action filtering 缩减动作空间；支持确定性回放。

## 主要贡献
- 2K+ 真实网站任务及逐步动作标注
- 语义 DOM filtering：把 HTML 元素空间缩减到 top-K 候选
- 确定性状态回放用于 offline 训练
- Web agent 的标准预训练数据
