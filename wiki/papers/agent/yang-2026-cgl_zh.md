# CGL: Advancing Continual GUI Learning via Reinforcement Fine-Tuning

## TL;DR（一句话总结）
> CGL：结合渐进式课程阶段和 RFT 的持续 GUI 学习框架；通过弹性权重巩固和回放机制在适应新 GUI 环境的同时防止灾难性遗忘。

## 主要贡献
- 渐进式课程：跨持续 GUI 环境的阶段性难度递增
- 弹性权重巩固 + 经验回放防止灾难性遗忘
- 带可验证 GUI 奖励的 RFT 实现高样本效率的持续适应
