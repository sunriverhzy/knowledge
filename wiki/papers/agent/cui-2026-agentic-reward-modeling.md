# Agentic Reward Modeling: Verifying GUI Agent via Online Proactive Interaction

## TL;DR
> VAGEN: replaces passive LLM judges with an active verifier agent that proactively probes the environment (executes commands, inspects state) to produce noise-free RLVR rewards for GUI agent training.

## Key Contributions
- VAGEN: verifier agent equipped with interaction tools to autonomously plan verification strategies
- Proactive environment probing: execute system commands to verify side-effects (not just observe screenshots)
- Enables noise-free RLVR for tasks without simple programmatic verifiers
- Addresses the limitation of passive LLM-as-judge for complex multi-step GUI verification
