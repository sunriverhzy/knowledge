# VisualWebArena: Evaluating Multimodal Agents on Realistic Visual Web Tasks

## TL;DR
> Visual extension of WebArena: 910 visual web tasks where correct action depends on visual understanding (image content, visual layout) not just text; tests visual grounding in web agents.

## Key Contributions
- 910 visual web tasks requiring visual understanding of page content
- Extends WebArena with image-dependent task specifications
- Key benchmark for multimodal web agent evaluation
