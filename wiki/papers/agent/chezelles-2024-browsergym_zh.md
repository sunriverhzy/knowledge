# The BrowserGym Ecosystem for Web Agent Research

## TL;DR（一句话总结）
> 聚合 WebArena、MiniWoB++、VisualWebArena、WebChoreArena 的统一 Gymnasium-style API；标准化 web-agent benchmark 基础设施。

## 主要贡献
- 面向 4+ web benchmark（WebArena、MiniWoB++、VWA、WebChoreArena）的统一 API
- 在标准接口背后处理 Docker、浏览器状态、DOM 提取
- 支持跨 benchmark 的 agent 训练与评测
