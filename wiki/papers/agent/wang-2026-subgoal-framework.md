# A Subgoal-driven Framework for Improving Long-horizon LLM Agents

## TL;DR
> Subgoal-driven framework (MiRA) for long-horizon LLM/GUI agents: turns abstract task into concrete subgoals with milestone-based planning and potential shaping rewards; directly applicable to web navigation.

## Key Contributions
- MiRA: milestone-based planning + potential shaping for long-horizon agent training
- Decomposes task into verifiable subgoals providing intermediate reward signal
- Potential shaping ensures reward consistency with original task objective
- Demonstrated on web navigation tasks requiring 20+ step planning
