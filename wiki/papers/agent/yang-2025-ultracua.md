# UltraCUA: A Foundation Model for Computer Use Agents with Hybrid Action

## TL;DR
> Foundation CUA that unifies visual GUI primitives and API/tool actions; trains routing between universal visual fallbacks and fast deterministic API executions.

## Key Contributions
- Unified hybrid action space: low-level GUI clicks + high-level API calls
- Learned routing between GUI and API modalities
- Trained on large-scale cross-platform computer-use data
