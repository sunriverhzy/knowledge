# Auto-scaling Continuous Memory for GUI Agent

## TL;DR
> Auto-scaling continuous memory for GUI agents: dynamically adjusts memory size based on task complexity; low-cost data flywheel via trajectory embedding and retrieval; enables reliable long-horizon GUI automation.

## Key Contributions
- Auto-scaling memory: dynamically resizes memory budget based on task complexity signals
- Continuous trajectory embedding: encodes GUI states and actions into a searchable memory
- Low-cost data flywheel: past successful trajectories improve future task performance
