# Magnet: Towards Adaptive GUI Agents with Memory-driven Knowledge Evolution

## TL;DR
> MAGNET: memory-driven GUI agent that maintains evolving knowledge across tasks; stationary memory (stable UI facts) + procedural memory (learned action patterns) enable adaptation to interface changes over time.

## Key Contributions
- Stationary memory: stores stable UI knowledge (element locations, button meanings) across sessions
- Procedural memory: stores learned action sequences for recurring task patterns
- Knowledge evolution: updates both memory types as interfaces change or new patterns are discovered
