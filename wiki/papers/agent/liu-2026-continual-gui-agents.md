# Continual GUI Agents

## TL;DR
> GUI-AiF: first continual learning framework for GUI agents using reinforcement fine-tuning; APR-iF and ARR-iF reward mechanisms stabilize grounding while preventing catastrophic forgetting as apps evolve.

## Key Contributions
- GUI-AiF: continual RL fine-tuning framework for GUI agents operating in dynamic environments
- APR-iF: reward encouraging diverse interaction points to prevent exploitation collapse
- ARR-iF: reward refining element regions to maintain grounding quality across updates
- First benchmark for evaluating GUI agent continual learning across evolving interfaces
