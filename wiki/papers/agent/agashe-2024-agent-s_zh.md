# Agent S: An Open Agentic Framework That Uses Computers Like a Human

## TL;DR（一句话总结）
> 开放式 computer-use agentic 框架；结合 GUI 交互、网页搜索和代码执行；从在线资源获取程序化记忆。

## 主要贡献
- 像人一样使用电脑的开放框架
- Experience-augmented 层次化规划：子任务级 Planner + GUI Executor
- 在线资源查询构建程序化记忆（类似人类查教程）
- 在 OSWorld 上有竞争力的性能
