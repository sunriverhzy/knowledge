# AgentCPM-GUI: Building Mobile-use Agents with Reinforcement Fine-tuning

## TL;DR（一句话总结）
> 带 RL fine-tuning 的 mobile-use agent：结合 SFT 预训练与真实 Android 任务上的 online RL；研究真实设备 I/O 噪声条件下的 RL。

## 主要贡献
- 真实 mobile 设备交互上的 RL fine-tuning（噪声环境）
- 奖励去噪：处理 RL 信号中的真实设备噪声
- 长时程 mobile 任务的上下文压缩
- EMNLP 2025 系统演示
