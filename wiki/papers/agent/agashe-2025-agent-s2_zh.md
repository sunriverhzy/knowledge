# Agent S2: A Compositional Generalist-Specialist Framework for Computer Use Agents

## TL;DR（一句话总结）
> 组合式 generalist-specialist CUA：通用规划器 + 针对不同 GUI 任务的专家执行器；OSWorld 和手机 benchmark SOTA。

## 主要贡献
- Generalist-specialist 分解：通用模型规划，专家模型执行特定领域 GUI 任务
- OSWorld 及手机 / desktop benchmark 强性能
- 模块化架构，允许热替换专家模块
