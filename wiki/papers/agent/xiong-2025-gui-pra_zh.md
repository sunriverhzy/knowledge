# GUI-PRA: Process Reward Agent for GUI Tasks

## TL;DR（一句话总结）
> GUI-PRA：GUI 任务过程奖励的裁判 agent；通过维护 GUI 状态变化和 UI 交互的动态记忆，克服标准 PRM 的'中间丢失'和 UI 状态变化盲区问题；ICLR 2026。

## 主要贡献
- GUI 状态变化的动态记忆：跟踪哪些 UI 元素发生了变化、出现或消失
- 克服'中间丢失'：PRM 关注所有步骤而非不成比例地加权早期/晚期步骤
- GUI 任务的过程奖励：每步的稠密反馈而非仅终止奖励
- ICLR 2026 接收
