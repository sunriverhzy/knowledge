# DynaWeb: Model-based Reinforcement Learning of Web Agents

## TL;DR
> First explicit Web World Model (WWM) for model-based RL: dream rollouts in learned web dynamics to bypass I/O wall; substantial gains on WebArena.

## Key Contributions
- WWM: neural model predicting next page state from current state + action
- Dream rollouts: 1000× throughput vs real-env interaction
- Periodic re-grounding on real environments to correct sim-to-real gap
- Substantial WebArena performance improvement
