# MobileGUI-RL: Advancing Mobile GUI Agent Through Reinforcement Learning in Online Environment

## TL;DR
> Scalable online RL framework for mobile GUI agents: GRPO with efficiency rewards trains agents to discover concise action paths; decouples CPU emulator from GPU inference for parallel training; ICLR 2026.

## Key Contributions
- Scalable online RL for mobile GUI: CPU emulator decoupled from GPU training for massive parallelism
- GRPO with efficiency rewards: penalize redundant actions, reward shorter successful paths
- Demonstrates that online RL can train production-quality mobile GUI agents from scratch
- ICLR 2026 accepted; strong performance on AndroidWorld
