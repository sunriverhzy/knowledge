# SeeClick: Harnessing GUI Grounding for Advanced Visual GUI Agents

## TL;DR
> GUI grounding as a standalone VLM capability: SeeClick fine-tunes VLMs on GUI grounding data; ScreenSpot benchmark; demonstrates grounding quality → task performance.

## Key Contributions
- Establishes GUI grounding as a core VLM capability (separate from task performance)
- ScreenSpot benchmark: IoU-based element localization across platforms
- Fine-tuning on grounding data improves downstream task performance
- Widely adopted as grounding evaluation baseline
