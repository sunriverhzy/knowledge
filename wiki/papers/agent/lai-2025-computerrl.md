# ComputerRL: Scaling End-to-end Online Reinforcement Learning for Computer Use Agents

## TL;DR
> End-to-end online RL for computer-use agents with API-GUI hybrid action space; introduces Entropulse to counter entropy collapse in long-horizon tasks.

## Key Contributions
- Formal API-GUI hybrid action paradigm: agents choose GUI coords vs API calls
- Entropulse: interleaved RL+SFT strategy to maintain policy entropy
- End-to-end online RL training on real computer environments
- Substantial performance gains on OSWorld over prior methods
