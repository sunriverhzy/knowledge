# UI-S1: Advancing GUI Automation via Semi-online Reinforcement Learning

## TL;DR
> Semi-online RL via trajectory patching: simulates online correction dynamics on offline data; Patch Module corrects OOD actions, dual-level advantage function.

## Key Contributions
- Patch Module: inserts corrective actions into offline trajectories to simulate OOD recovery
- Dual-level advantage: trajectory-level + action-level advantage estimation
- Achieves online-RL-like performance without real-env interaction cost
