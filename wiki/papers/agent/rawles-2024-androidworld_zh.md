# AndroidWorld: A Dynamic Benchmarking Environment for Autonomous Agents

## TL;DR（一句话总结）
> 动态 Android RL benchmark：116 个参数化任务模板生成数百万独特任务；通过 ADB / SQLite 状态检查提供真值奖励；防止死记硬背。

## 主要贡献
- 116 个参数化任务模板 → 数百万任务变体
- 直接 ADB 数据库查询的真值奖励（零噪声）
- 通过任务随机化防止过拟合
- Mobile RL 训练的权威'Gymnasium'
