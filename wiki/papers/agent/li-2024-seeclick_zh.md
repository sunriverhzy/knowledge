# SeeClick: Harnessing GUI Grounding for Advanced Visual GUI Agents

## TL;DR（一句话总结）
> GUI grounding 作为独立 VLM 能力：SeeClick 在 GUI grounding 数据上微调 VLM；ScreenSpot benchmark；证明 grounding 质量 → 任务性能。

- 被广泛用作 grounding 评测基线
