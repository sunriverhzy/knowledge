# OS-ATLAS: A Foundation Action Model for Generalist GUI Agents

## TL;DR
> Large-scale GUI action foundation model with 13M training examples; strong grounding and action generation baseline for cross-platform GUI agents.

## Key Contributions
- 13M GUI action examples across web, mobile, desktop
- Strong baseline on grounding benchmarks (ScreenSpot, etc.)
- Foundation for downstream GUI RL fine-tuning
