# GUI-Eyes: Tool-augmented Perception for Visual Grounding in GUI Agents

## TL;DR
> Active perception for GUI grounding: agent autonomously invokes visual zoom/crop tools before predicting coordinates; GRPO training; Qwen3-VL backbone.

## Key Contributions
- Active perception: agent decides when to invoke zoom tool before grounding
- Tool-augmented visual pipeline: zoom patches provide higher resolution for small elements
- GRPO training on grounding tasks with tool invocation rewards
- Cross-platform (D/M/W); Qwen3-VL backbone
