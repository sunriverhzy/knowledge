# Hi-Agent: Hierarchical Vision-Language Agents for Mobile Device Control

## TL;DR（一句话总结）
> 层次化 Planner+Executor 架构：Planner 生成语义子目标，Executor 执行 UI 动作；GRPO + Foresight Advantage Function；AitW 87.9%。

## 主要贡献
- Planner：从高层任务描述生成语义子目标
- Executor：在当前子目标引导下执行原子 UI 动作
- Foresight Advantage Function：advantage 估计中考虑未来任务进度
- 在 AndroidInTheWild 上 87.9%（发布时 SOTA）
