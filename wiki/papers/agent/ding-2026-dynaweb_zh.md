# DynaWeb: Model-based Reinforcement Learning of Web Agents

## TL;DR（一句话总结）
> 第一个用于 model-based RL 的显式 Web World Model（WWM）：在学得的 web 动态中做 dream rollout，绕过 I/O wall；在 WebArena 上有显著提升。

## 主要贡献
- WWM：从当前状态 + 动作预测下一页面状态的神经网络
- Dream rollout：相比真实环境交互 1000 倍吞吐量
- 定期在真实环境上重新对齐以修正 sim-to-real gap
- WebArena 上有显著性能提升
