# M2-Miner: Multi-Agent Enhanced MCTS for Mobile GUI Agent Data Mining

## TL;DR
> M2-Miner: first MCTS-based automated framework for mobile GUI agent data mining; InferAgent + OrchestraAgent + JudgeAgent collaborate to mine high-quality trajectories; 64× efficiency improvement; ICLR 2026.

## Key Contributions
- MCTS-guided trajectory exploration: systematic coverage of app state space
- Three-agent collaboration: InferAgent (inference) + OrchestraAgent (coordination) + JudgeAgent (quality filtering)
- Intent recycling: reuses discovered intents to reduce redundant exploration
- 64× mining efficiency improvement over prior methods; ICLR 2026
