# Ferret-UI: Grounded Mobile UI Understanding with Multimodal LLMs

## TL;DR
> Mobile UI grounding with any-resolution adaptability: handles elongated mobile screenshot aspect ratios; fine-grained annotations enabling Visual CoT; reduces hallucination in RL exploration.

## Key Contributions
- Any-resolution adaptability: handles elongated mobile screenshot aspect ratios (not 224×224)
- Fine-grained regional annotations for mobile UI elements
- Visual CoT: verbalize spatial reasoning before acting — reduces hallucination
- Used for anti-hallucination reward shaping in mobile GUI RL
