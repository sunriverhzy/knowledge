# Mind2Web: Towards a Generalist Agent for the Web

## TL;DR
> Web agent dataset + model: 2K+ diverse web tasks with action annotations; semantic DOM action filtering reduces action space; enables deterministic replay.

## Key Contributions
- 2K+ real-website tasks with step-by-step action annotation
- Semantic DOM filtering: reduces HTML element space to top-K candidates
- Deterministic state replay for offline training
- Standard pretraining data for web agents
