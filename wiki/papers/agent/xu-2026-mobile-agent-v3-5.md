# Mobile-Agent-v3.5: Multi-platform Fundamental GUI Agents

## TL;DR
> GUI-Owl-1.5: multi-platform GUI agent (2B–235B) with Hybrid Data Flywheel, MRPO (multi-platform RL), tool/MCP use, and memory; SOTA on 20+ benchmarks including OSWorld 56.5%, AndroidWorld 71.6%, ScreenSpotPro 80.3%.

## Key Contributions
- GUI-Owl-1.5: instruct/thinking model family from 2B to 235B; cloud-edge collaboration
- Hybrid Data Flywheel: simulated environment + cloud sandbox for UI understanding and trajectory generation
- MRPO: multi-platform RL algorithm mitigating platform conflicts and long-horizon training inefficiency
- OSWorld 56.5%, AndroidWorld 71.6%, WebArena 48.4%, ScreenSpotPro 80.3%, OSWorld-MCP 47.6%
