# UI-R1: Enhancing Efficient Action Prediction of GUI Agents by Reinforcement Learning

## TL;DR
> Rule-based RLVR with DAST (Difficulty-Adaptive Slow-Thinking): adapts reasoning depth to task difficulty; cross-platform GUI grounding and action.

## Key Contributions
- DAST: difficulty-adaptive thinking — complex tasks get more CoT tokens
- Rule-based RL rewards: action-type verification + format compliance
- Cross-platform (D/M/W) GUI grounding and multi-step task execution
- 3B model size; strong data efficiency
