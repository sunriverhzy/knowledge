# MMBench-GUI: Hierarchical Multi-Platform Evaluation Framework for GUI Agents

## TL;DR
> MMBench-GUI: hierarchical benchmark spanning 6 platforms (Windows/macOS/Linux/iOS/Android/Web) and 4 capability levels (Content Understanding, Grounding, Task Automation, Task Collaboration); ICLR 2026.

## Key Contributions
- 6-platform benchmark: Windows, macOS, Linux, iOS, Android, Web in one framework
- 4 hierarchical levels: Content Understanding → Element Grounding → Task Automation → Task Collaboration
- Standardized evaluation toolbox for GUI agent comparison
- ICLR 2026 accepted
