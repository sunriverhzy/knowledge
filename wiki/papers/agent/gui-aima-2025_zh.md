# GUI-AIMA: Aligning Intrinsic Multimodal Attention with a Context Anchor for GUI Grounding

## TL;DR（一句话总结）
> GUI-AIMA：通过将模型内在多模态注意力与上下文锚点对齐实现无坐标 grounding；提取模型正在关注的图像 patch 而非预测显式坐标。

## 主要贡献
- 仅注意力 grounding：无坐标 token 预测；使用内在模型注意力
- 上下文锚点：参考元素引导空间注意力校准
- 无坐标：自然处理变分辨率和变化的 UI 布局
