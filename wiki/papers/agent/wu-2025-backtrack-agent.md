# BacktrackAgent: Enhancing GUI Agent with Error Detection and Backtracking Mechanism

## TL;DR
> Generator-Verifier-Judger-Reflector architecture: detects errors, backtracks to safe state, retries with improved strategy; EMNLP 2025.

## Key Contributions
- Error detection: Verifier checks if current state matches expected state
- Backtracking: undo recent actions to last known-good state
- Reflector: update strategy based on failed attempt information
- EMNLP 2025; cross-platform GUI tasks
