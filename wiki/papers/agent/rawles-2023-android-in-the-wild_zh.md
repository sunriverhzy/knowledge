# Android in the Wild: A Large-scale Dataset for Android Device Control

## TL;DR（一句话总结）
> 715K+ 跨多样应用和任务的 Android 设备交互轨迹；用于 offline GUI agent 预训练和 offline RL 初始化（DigiRL、Digi-Q）。

## 主要贡献
- 715K+ 来自真实 Android 设备使用的交互轨迹
- 多样应用、任务和跨版本 UI 布局
- Mobile GUI agent 预训练的标准数据（DigiRL、Digi-Q）
