# AgentPRM: Process Reward Models for LLM Agents via Step-wise Promise and Progress

## TL;DR
> AgentPRM: process reward model for LLM agents using Promise (potential to succeed) and Progress (state improvement) signals at each step; provides dense credit assignment for multi-step GUI agent training.

## Key Contributions
- Promise signal: estimates probability of future task success from current step
- Progress signal: measures improvement in task state compared to previous step
- Combines both for dense credit assignment without requiring full trajectory outcomes
