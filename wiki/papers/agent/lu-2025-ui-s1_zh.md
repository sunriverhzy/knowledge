# UI-S1: Advancing GUI Automation via Semi-online Reinforcement Learning

## TL;DR（一句话总结）
> 通过轨迹 patching 的 semi-online RL：在离线数据上模拟在线纠错动态；Patch Module 纠正 OOD 动作，双层 advantage 函数。

## 主要贡献
- Patch Module：在离线轨迹中插入纠正动作，模拟 OOD 恢复
- 双层 advantage：轨迹级 + 动作级 advantage 估计
- 在不付出真实环境交互代价的情况下达到近在线 RL 性能
