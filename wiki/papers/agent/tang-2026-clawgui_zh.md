# ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents

## TL;DR（一句话总结）
> ClawGUI：GUI agent 生命周期的端到端统一框架，覆盖多平台多应用的训练流水线、评测工具和部署基础设施。

## 主要贡献
- 统一框架：训练、评测和部署 GUI agent 的单一代码库
- 多平台支持：单一训练/评测流水线内覆盖 web、desktop、mobile
- 标准化从数据到部署的 GUI agent 开发工作流
