# Continual GUI Agents

## TL;DR（一句话总结）
> GUI-AiF：第一个面向 GUI agent 的持续学习框架，使用强化微调；APR-iF 和 ARR-iF 奖励机制在应用演化时稳定 grounding 并防止灾难性遗忘。

## 主要贡献
- GUI-AiF：面向动态环境 GUI agent 的持续 RL 微调框架
- APR-iF：奖励多样化交互点，防止利用坍缩
- ARR-iF：奖励细化元素区域，跨更新维持 grounding 质量
- 第一个评测 GUI agent 跨演化界面持续学习的 benchmark
