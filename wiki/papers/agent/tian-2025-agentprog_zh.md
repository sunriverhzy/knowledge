# AgentProg: Empowering Long-horizon GUI Agents with Program-guided Context Management

## TL;DR（一句话总结）
> AgentProg：长时程 GUI agent 的程序引导上下文管理；使用可执行程序将多步历史压缩并组织成跨交互持续的结构化记忆。

## 主要贡献
- 程序引导上下文：agent 编写迷你程序来摘要和压缩交互历史
- 可执行记忆：存储的程序可重新执行以恢复任务上下文
- 在保留任务关键信息的同时显著减少上下文窗口使用
