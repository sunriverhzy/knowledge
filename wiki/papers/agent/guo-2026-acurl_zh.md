# Autonomous Continual Learning of Computer-Use Agents for Environment Adaptation

## TL;DR（一句话总结）
> ACuRL：自主课程 RL 框架，零人工数据地持续将 computer-use agent 适配到新环境；多阶段流水线：探索、上下文回顾、能力评估、课程任务生成、RL 训练。

## 主要贡献
- ACuRL：CUA 环境适应的多阶段自主课程 RL
- 零人工数据：通过环境探索和能力评估全自主生成课程
- 处理应用演化/更新或 agent 遇到新软件时的分布偏移
- 直接解决真实部署挑战：agent 必须在用户特定环境中工作
