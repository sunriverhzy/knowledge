# OS-HARM: A Benchmark for Measuring Safety of Computer Use Agents

## TL;DR（一句话总结）
> OS-HARM：第一个全面的 computer-use agent 安全 benchmark；衡量真实 OS 任务中意外动作（数据删除、非预期购买）和对抗性 UI 攻击造成的危害。

## 主要贡献
- 第一个专门面向真实 OS 环境 computer-use agent 的安全 benchmark
- 衡量意外危害：agent 错误导致的数据丢失、财务动作、隐私侵犯
- 对抗性危害：利用 agent 漏洞的 UI 注入攻击、恶意弹窗
- 提供 CUA 失败模式分类法和安全评测方法论
