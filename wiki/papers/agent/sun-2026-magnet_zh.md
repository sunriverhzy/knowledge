# Magnet: Towards Adaptive GUI Agents with Memory-driven Knowledge Evolution

## TL;DR（一句话总结）
> MAGNET：记忆驱动的 GUI agent，跨任务维护演化的知识；静态记忆（稳定 UI 事实）+ 程序性记忆（学习的动作模式）使 agent 能随时间适应界面变化。

## 主要贡献
- 静态记忆：跨会话存储稳定的 UI 知识（元素位置、按钮含义）
- 程序性记忆：存储反复任务模式的学习动作序列
- 知识演化：随界面变化或发现新模式时更新两种记忆类型
