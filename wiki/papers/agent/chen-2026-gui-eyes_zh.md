# GUI-Eyes: Tool-augmented Perception for Visual Grounding in GUI Agents

## TL;DR（一句话总结）
> GUI grounding 的 active perception：agent 在预测坐标前自主调用视觉缩放 / 裁剪工具；GRPO 训练；Qwen3-VL backbone。

## 主要贡献
- Active perception：agent 在 grounding 前决定何时调用缩放工具
- Tool-augmented 视觉管线：zoom patch 为小元素提供更高分辨率
- grounding 任务上带 tool 调用奖励的 GRPO 训练
- 跨平台（D/M/W）；Qwen3-VL backbone
