# GUI-R1: A Generalist R1-style Vision-Language Action Model for GUI Agents

## TL;DR（一句话总结）
> GUI agent 的 R1-style RLVR+GRPO：统一动作空间 + 二元可验证奖励；仅 3K 训练样本在 8 个 benchmark 上 SOTA；涌现思维链推理。

## 主要贡献
- 统一 GUI 动作空间：点击、滑动、键盘统一表示
- 二元可验证奖励：坐标在 BBox 内 +1，否则 0
- GUI-R1-3K：仅 3K 精选样本 → ScreenSpot-Pro + 7 benchmark SOTA
- 涌现 System-2 推理：无显式监督的自发内部独白
