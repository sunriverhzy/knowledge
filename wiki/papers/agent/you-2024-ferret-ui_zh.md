# Ferret-UI: Grounded Mobile UI Understanding with Multimodal LLMs

## TL;DR（一句话总结）
> 带任意分辨率适配的 mobile UI grounding：处理纵向移动截图长宽比；细粒度标注支持 Visual CoT；减少 RL 探索中的 hallucination。

## 主要贡献
- 任意分辨率适配：处理纵向移动截图长宽比（非 224×224）
- 移动 UI 元素的细粒度区域标注
- Visual CoT：行动前口述空间推理——减少 hallucination
- 用于 mobile GUI RL 的抗 hallucination 奖励塑造
