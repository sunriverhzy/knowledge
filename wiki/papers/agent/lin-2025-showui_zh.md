# ShowUI: One Vision-Language-Action Model for GUI Visual Agent

## TL;DR（一句话总结）
> 面向 GUI agent 的统一 VLA（Vision-Language-Action）模型；交错的 UI-guided 视觉 token 选择在保持 grounding 精度的同时减少计算量。

## 主要贡献
- 单一模型同时处理 grounding、规划和动作生成
- UI-guided 视觉 token 选择：聚焦相关 UI 区域
- 在 ScreenSpot 及 GUI 任务 benchmark 上强性能
