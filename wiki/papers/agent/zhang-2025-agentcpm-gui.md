# AgentCPM-GUI: Building Mobile-use Agents with Reinforcement Fine-tuning

## TL;DR
> Mobile-use agent with RL fine-tuning: combines SFT pretraining with online RL on real Android tasks; studies RL under noisy real-device I/O conditions.

## Key Contributions
- RL fine-tuning on real mobile device interactions (noisy environment)
- Reward denoising for handling real-device noise in RL signals
- Context compression for long-horizon mobile tasks
- EMNLP 2025 system demonstration
