# Co-EPG: A Framework for Co-evolution of Planning and Grounding in Autonomous GUI Agents

## TL;DR
> Co-EPG: jointly trains GUI planning and grounding modules that co-evolve, where planning improvements guide better grounding targets and accurate grounding enables more reliable planning.

## Key Contributions
- Co-evolutionary training: planning and grounding mutually improve each other
- Planning-to-grounding signal: planner provides context that helps localize target elements
- Grounding-to-planning signal: precise grounding enables more reliable plan execution
