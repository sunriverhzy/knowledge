# ScreenAgent: A Vision Language Model-driven Computer Control Agent

## TL;DR（一句话总结）
> 通过 VNC pixel-stream 的 VLM 驱动桌面 computer control；Plan-Act-Reflect 自纠错循环；不依赖 accessibility API 的跨平台方法。

## 主要贡献
- 通过 VNC 的纯 pixel-stream：无 accessibility API 依赖
- Plan-Act-Reflect 循环：反思动作结果以自我纠正
- 跨平台：适用于任何暴露 VNC 接口的 OS
