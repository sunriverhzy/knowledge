## 本 Wiki 的覆盖范围

本 Wiki 聚焦于以下交叉领域的研究：

- **大语言模型（LLM）**
- **强化学习（RL）**，尤其是应用于 LLM 对齐和 Agent 训练的部分
- **AI 生成内容（AIGC）**，包括扩散模型与水印
- **多模态学习**，尤其是视觉语言模型（VLM）与视频理解
- **基于 LLM 的 Agent**，包括多智能体系统与 GUI Agent
- **奖励建模与对齐**

纯 CV、纯 NLP、纯 RL 理论等周边话题，仅在与上述主线直接相关时才纳入。
